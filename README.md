# Punjab District Development Dashboard

## Live Demo

https://punjab-district-development-dashboard.streamlit.app/

## Project Overview

The Punjab District Development Dashboard is an interactive data analytics platform developed to support evidence based development planning across the districts of Punjab, Pakistan.

The dashboard enables users to explore demographic, educational, healthcare, and infrastructure indicators through interactive visualizations and automated reporting.
